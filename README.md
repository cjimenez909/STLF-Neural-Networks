# STLF-Neural-Networks
Short-term load forecasting (STLF) is the process of predicting electricity demand over a certain time period. I performed STLF using a dataset on 30 homes in Ontario, Canada, using Python, Jupyter Notebooks, PyTorch, and Matplotlib. I gained experience with tuning model parameters and trained my neural network on GPUs with the Perlmutter supercomputer. I completed and presented this project during Intro to HPC Bootcamp in August 2025.
